# 주제: OverTheWire Bandit (Level 10 → 11)

## 1. 문제 목표
data.txt 파일에 비밀번호가 들어있는데, 이 내용이 Base64로 인코딩(Encoded)되어 있다고 함.
이걸 원래대로 해독(Decode)해서 비밀번호를 찾아야 함.

## 2. 접근 방법
처음에 'Base64'라는 단어를 보고 이게 뭔지 몰라서 당황함.
그래서 문제 풀기 전에 Base64가 무엇인지부터 구글링해서 알아봄.
컴퓨터 데이터를 문자열로 바꾸는 방식이라는 걸 알게 되었고, 리눅스에는 이걸 해독해 주는 명령어가 따로 있다는 것을 찾아서 적용함.

## 3. 풀이 명령어
```bash
base64 -d data.txt
