# 주제: OverTheWire Bandit (Level 11 → 12) 

---

## 1. 문제 목표
data.txt 파일의 내용이 모든 알파벳(대소문자)이 13글자씩 밀려있는 상태로 저장되어 있음.
이걸 원래대로 복구해서 비밀번호를 찾아야 함.

## 2. 접근 방법
문제를 읽자마자 예전에 들어본 '시저 암호(Caesar Cipher)'가 떠올랐음.
알파벳 순서를 일정 간격만큼 밀어서 글자를 바꾸는 방식인데, 이번 문제는 정확히 13칸을 미는 'ROT13' 방식이었음.
일일이 손으로 바꿀 수는 없으니 리눅스에서 문자를 교환해 주는 명령어를 찾아봄.

## 3. 풀이 명령어
```bash
cat data.txt | tr 'a-zA-Z' 'n-za-mN-ZA-M'
