# 주제: OverTheWire Bandit (Level 14 → 15)

---

## 1. 문제 목표
현재 레벨(bandit14)의 비밀번호를 localhost의 30000번 포트(Port)로 보내면, 다음 레벨의 비밀번호를 알려준다고 함.

## 2. 접근 방법
지금까지는 파일이나 명령어로 비밀번호를 찾았는데, 이번에는 '네트워크 통신'을 해야 함.
서버의 특정 포트에 접속해서 데이터를 주고받을 수 있는 도구가 필요함.
리눅스에서 '네트워크의 맥가이버 칼'이라고 불리는 nc(Netcat) 명령어나 telnet을 사용하기로 함.

## 3. 풀이 명령어
```bash
# 1. 현재 비밀번호 확인 (이전 레벨에서 얻은 경로)
cat /etc/bandit_pass/bandit14

# 2. nc 명령어로 30000번 포트에 접속 후 비밀번호 입력
nc localhost 30000
# (위에서 확인한 비밀번호 붙여넣기) -> 정답 출력됨
